# AI Literacy Curriculum — SUNY

Working materials of an AI literacy curriculum developed at SUNY Polytechnic Institute. Two scaffolds are in use, and the conversations that produced them are archived alongside.

## What is here

- `index.html` — front page
- `ai-literacy-learners-permit.html` — the **Learner's Permit**, a content-neutral 15-exercise scaffold for foundational generative AI literacy
- `ai-literacy-commercial-license.html` — the **Commercial License**, a 15-exercise scaffold for agentic AI operation
- `archive/` — chronological record of the conversations that produced the two scaffolds, one HTML page per conversation, plus an index

The premise: the transcript is the work; the artifact is evidence the work concluded. The archive preserves the work.

## What has been redacted

Personal references have been removed from the archive before publication:

- **Names** of collaborators, students, and others mentioned in the conversations
- **Email addresses** in any form
- **Email-message headers** (From, To, Subject, Date, Sent, etc.)

Redactions appear as `[redacted]` in running text, `[redacted email]` for addresses, and `[redacted header]` for stripped header lines. The structure of each exchange is preserved so the cognitive arc of the conversation remains visible.

The author's own name is retained.

## What has been kept

The unredacted prompt-and-response sequence of each conversation, including reasoning, corrections, false starts, and structural decisions. The conversations were treated as primary source material from the outset; this archive treats them the same way, with the privacy of named individuals protected.

## Using this resource

The HTML files and transcript `.md` files can be uploaded to any LLM to generate your own curriculum variants or exercises using these scaffolds. The resulting HTML artifacts can be uploaded directly to Brightspace or equivalent LMS.

Hosted version: [stevesunypoly.github.io/ai-literacy-2026/](https://stevesunypoly.github.io/ai-literacy-2026/)

## Source materials

The original Claude conversation exports (JSON), the redaction list, and the build script live outside this repository on the maintainer's local machine. They are excluded by `.gitignore` and are not distributed. Re-running the build requires those local materials.
